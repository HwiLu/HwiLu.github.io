强行校正服务器时间
使用以下格式：
```
date -s 14:22:20
```
如果有相应的时钟源，也可以使用`ntpdate`进行同步
```
# ntpdate  0.cn.pool.ntp.org

# date
```
查看硬件时钟：
```
# hwclock --show
```
然后把系统时间写入到硬件：
```
# hwclock -w
# hwclock --show
```
